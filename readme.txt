出退勤ボタン2.0
本アプリケーションはつばさで働く上で必ず提出する出勤退勤メール等
を簡略化するために作ったアプリケーションです。
作業の簡略化に繋がれば幸いです。

利用者登録画面（初回時のみ）
　　　　初回のみ名前入力画面が出るので名前を入力してください。
　　　　入力した名前がメールに反映されます。
　　　
メインウィンドウ
　　　1レポート送信
　　　　レポートを送る際のテンプレートを呼び出します。
　　　　アカウント設定は不要です
　　　2メール送信
　　　　送信するメールの設定を開きます。
　　　3ジョブカンボタン
　　　　ジョブカンに自動でログインします
　　　　アカウントの設定とドライバーの更新をすることで自動ログインができるようになります
     4簡易マニュアル
　　　　本アプリケーションを初めて開いたときに表示されるマニュアルを表示します
　　　5アカウント設定
　　　　アカウント設定ボタンを開きます。　　
　　　6ドライバー更新
　　　　ジョブカンの自動ログインする為に必要なドライバーをインストールできます。
　　　　chromeドライバーをバージョン～に更新しましたという文章が出てきたらインストール完了です。

サブウィンドウ
　　　1設定（アカウント設定）
　　　　outlookメールやジョブカンのアカウントでログインする為の設定をします。
　　　　ご自身のメールアドレス、ジョブカンのパスワード、MicrosoftOfficeのpasswordを設定してください。
　　　2メール送信
　　　　コンボボックスを開くと出勤、退勤、日報、の項目が出てくるので、
              送りたいメールの種類を選んでください。
　　　　項目を選択すると宛先、件名、本文が表示されます。
　　　　本文は必要に応じて変更してください。
　　　　変更した内容は戻るボタンか送信ボタンを押すと反映されます。。
